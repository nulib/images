CanonCustom Functions2 Tags

Beginning with the EOS 1D Mark III, Canon finally created a set of custom function tags which are consistent across models. The EOS 1D Mark III has 57 custom function tags divided into four main groups: 1. Exposure (0x0101-0x010f), 2. Image (0x0201-0x0203), Flash Exposure (0x0304-0x0306) and Display (0x0407-0x0409), 3. Auto Focus (0x0501-0x050e) and Drive (0x060f-0x0611), and 4. Operation (0x0701-0x070a) and Others (0x080b-0x0810). The table below lists tags used by the EOS 1D Mark III, as well as newer values and tags used by the EOS 40D. It is expected that future models will re-use some of these tags (possibly with minor value changes), as well as defining additional new tags of their own.
