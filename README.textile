h2. Developer prereqs

rvm 1.10 or later
git 1.7 or later

h2. Installation

h3. Clone & migrate

<pre>
git clone --recursive <git repo URL>
bundle install
rake db:migrate
</pre>

h3. Configuration

rake jetty:config

h2. To run the application

<pre>
rake jetty:config
rake jetty:start
rake hydra:fixtures:refresh
unicorn_rails
</pre>

The server will be started on port 8080 by default.


h2. testing the application

<pre>
rake hydra:fixtures:refresh RAILS_ENV=test
rake spec
</pre>

h2. Local development only

You'll need to install imagemagick to handle the tiff to jp2 conversion on your local machine. 
brew install imagemagick --with-libtiff --with-jp2
